# strtotime()

将任何字符串的日期时间描述解析为 Unix 时间戳

strtotime("2018-01-18 08:08:08");  // 将指定日期转成时间戳

# PHP Misc. 杂项函数

# uniqid()

uniqid() 函数基于以微秒计的当前时间，生成一个唯一的 ID。

**注释：**由于基于系统时间，通过该函数生成的 ID 不是最佳的。如需生成绝对唯一的 ID，请使用 md5()open in new window 函数。
